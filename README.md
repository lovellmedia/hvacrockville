# HVAC Rockville

Astro conversion of the Kimi HTML microsite template for hvacrockville.com.

## Folders

- `src/pages/`: Astro route files converted from the Kimi HTML template.
- `src/components/`: Shared SEO head component used by every page.
- `src/data/`: Image manifest and structured site data.
- `public/`: Static CSS, JavaScript, generated image assets, and favicon.

## Commands

- `npm install`
- `npm run build`
- `npm run dev`
